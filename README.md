# Alin
Alin is a file friendly framework that lets you do HTML CSS and JS in one javascript file with more simplicity!

To get started, you must load the system.js file into an html file, there was no other way to make it fully independent and run on the web
and then once you do that, make a javascript file and start making whatever you want! to make these take effect, simply load the script you made into the html file, thats it!

if you want to learn the syntax of the programming language, you can either check the system.js file or wait for a documentation on how to use this framework!
